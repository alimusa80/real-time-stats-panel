# Security Policy

If you discover a security vulnerability in `alimusa/real-time-stats-panel`, please do not open a public issue first.

Report it privately to:

- Website: https://www.alimusa.so/
- GitHub: https://github.com/alimusa80

Include:

- affected version
- reproduction steps
- impact summary
- any suggested mitigation

You will receive confirmation as soon as the report is reviewed.
