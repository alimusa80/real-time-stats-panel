# Changelog

All notable changes to `alimusa/real-time-stats-panel` will be documented in this file.

## 1.0.0 - 2026-03-25

- Initial release of the `RealTimeStatsPanel` Filament plugin
- Added real-time dashboard widget with Echo-based updates and sparkline cards
- Added cache-backed stats manager for active users, request rate, CPU, and memory metrics
- Added private broadcasting channels, events, listeners, and panel authorization hooks
- Added simulation command, package metadata, artwork, and distribution-ready documentation
